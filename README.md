## 📋 معرفی

**𝔸𝕪𝕙𝕒𝕟𝕏-𝔽𝕣𝕖𝕕𝕠𝕞-𝕊𝕔𝕒𝕟𝕟𝕖𝕣-🧑‍💻🌿** یک ابزار قدرتمند Python با عملکرد بالا است که برای اسکن رنج‌های امن IP طراحی شده است.

این ابزار فرآیند ایجاد تونل‌های متمرکز بر حریم خصوصی را با انتخاب خودکار نقاط پایانی از رنج‌های IP قابل اعتماد و پرسرعت (مانند **ArvanCloud** و **Cloudflare**) ساده می‌کند. این کار باعث پنهان‌سازی ترافیک می‌شود و ترافیک تونل شما را مانند ترافیک عادی HTTPS نشان می‌دهد تا محدودیت‌ها دور زده شده و ناشناس بودن افزایش یابد.
